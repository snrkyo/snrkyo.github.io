常用工具合集

FSCapture

一个绿色软件，主要用于微信公众稿的长截图，其它功能尚待发现。

下载地址

Typora

一个轻量级文本编辑器，支持 Markdown 语言，实时渲染。

官方网址

Guetzli

Google 最新开源的一个图片压缩算法，具有 30% 左右的压缩率。对某些图片具有较好的压缩率。运行会消耗大量内存和时间。

Github

KillMir 1.0

一个病毒专杀程序，西活楼下打印店病毒。该病毒会在 U盘里写入与文件夹同名的副本，并修改图标为Win7系统的文件夹图标，具有一定的隐藏性，危害性暂时不明，只知道可以自我复制并驻留内存。

Basic

An interpreter for untyped lambda calculus, beta-reduction.
